# Matemática Financeira — Notas de Aula

Notas de aula da disciplina **Matemática Financeira** do curso de Graduação em Economia.

## 📄 Material

O PDF com todas as notas está disponível diretamente neste repositório:

👉 **[MatFinanceira_NotasDeAula.pdf](./MatFinanceira_NotasDeAula.pdf)**

Clique em *"Download raw file"* (ícone de download no canto superior direito) para salvar no seu computador.

## 📚 Conteúdo

| Capítulo | Tema |
|----------|------|
| Cap. 1 | Regimes de Capitalização — juros simples, compostos e contínuos |
| Cap. 2 | Séries Finitas Periódicas — anuidades, gradientes e séries em PG |
| Cap. 3 | Séries com Capitalização Instantânea |
| Cap. 4 | Séries Infinitas — perpetuidades e Modelo de Gordon |
| Cap. 5 | Sistemas de Amortização — PRICE, SAC e Sistema Americano |
| Cap. 6 | Formas de Cotar Taxa — nominal, efetiva, real e equação de Fisher |
| Cap. 7 | Análise de Investimentos — VPL, TIR e TIRM |
| Cap. 8 | Títulos Públicos — LFT, LTN, NTN-B e NTN-F |

## 🛠️ Compilar o LaTeX (opcional)

Os arquivos-fonte em LaTeX estão disponíveis para quem quiser compilar localmente. É necessário ter uma distribuição LaTeX instalada (recomenda-se [TeX Live](https://www.tug.org/texlive/) ou [MiKTeX](https://miktex.org/)).

```bash
# Na pasta do repositório, execute duas vezes para gerar referências cruzadas:
pdflatex main.tex
pdflatex main.tex
```

### Estrutura dos arquivos

```
├── main.tex          # Documento mestre
├── preambulo.tex     # Pacotes e ambientes customizados
├── cap01.tex         # Capítulo 1
├── cap02.tex         # Capítulo 2
│   ...
└── cap08.tex         # Capítulo 8
```
